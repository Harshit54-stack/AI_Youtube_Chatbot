# ─────────────────────────────────────────────────────────────────────────────

# Install:   pip install -r backend/requirements.txt
# Upgrade:   pip install -r backend/requirements.txt --upgrade
#
# Notes
# -----
# * youtube-transcript-api has been REMOVED — replaced by Supadata REST API.
#   Supadata works reliably on cloud providers (Render, Railway) without the
#   IP-blocking issues that affect direct YouTube scraping.
# * langchain-google-genai provides ChatGoogleGenerativeAI for Gemini access.
# * All LangChain packages are pinned to the same major version for
#   compatibility. Keep them in sync when upgrading.
# * faiss-cpu is used for deployment; switch to faiss-gpu locally if you
#   have a CUDA-capable GPU.
# ─────────────────────────────────────────────────────────────────────────────
